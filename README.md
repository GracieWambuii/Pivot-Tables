# Pivot-Tables
- Pivot Tables are tools used to analyze, explore, group and present largedatasets.
- The above pivot table was created to explore and analyze bike sales in Europe. I sorted and filtered the dataset according     to Country, cost, Revenue and profit. 
- This allowed me to compare which country and state brought in more revenue.
  
 # Benefit of Pivot tables: It is a time saving tool, I was able to instantly summarize, analyze and reorganize a large dataset without complex formulas
